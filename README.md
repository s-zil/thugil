This repository contains a Preprocessor Directive or header file called `thugil.h` It is written in pure C programming language. 
Download the repo in a folder or copy-paste this line that will create a folder called `thugil`: 

`git clone thugil https://github.com/s-zil/thugil.git`

Nagivate into `thugil` folder, and write a source code to test the utility macros in this header file. Or simply use
the `kisinna.c` program to test the Utility Macros. 

These macros are related to long quotations translated into Zou language from English-language souces. 
Some quotations are original composition in Zou and translated into English. More quotations will be added in the future. 
As of July 2026, there are only two utility macros in `thugil.h`
    
    ///LEIGOU
    This utility macro relates to the Lockean idea of how property (especially landed property) is 
    a foundational priciple of building and sustaining a stable social order. It is an original composition in Zou 
    and translated into English.
    
    ///SUAHLAM
    This a fragment of a poem written by T.S. Eliot ("Journey of the Magi") translated into Zou language
    by David Vumlallian Zou. 

//DUAL PURPOSE

This repository has a dual purpose: 

(a) It will make learning C programming, especially Macro feature of header files 
in C programming, a fun activity for low-resourced languages such as the Zou. 

(b) The Macro Utilities in `thugil` repository such as LEIGOU, SUAHLAM, etc will become valuable linguistic resources 
for students of C language.
