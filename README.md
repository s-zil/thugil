This repository `thugil` contains a Preprocessor Directive or header file called `thugil.h` 
written in pure C programming language. Downloaded in a folder or git clone the repository URL.
Inside that folder, write a source code to use the utility macros in this header file. Or simply use
the kisinna.c program to test the Utility Macros related to long quotations translated into Zou language
from English-language souces. Some quotations are original composition in Zou and translated into English.
More quotations will be added in the future. As of July 2026, there are only two utility macros in `thugil.h`
    
    LEIGOU: This utility macro relates to the Lockean idea of how property (especially landed property) is 
    a foundational priciple of building and sustaining a stable social order. It is an original composition in Zou 
    and translated into English.
    
    SUAHLAM: This a fragment of a poem written by T.S. Eliot ("Journey of the Magi") translated into Zou language
    by David Vumlallian Zou.
